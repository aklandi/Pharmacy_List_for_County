# Pharmacy List for County

In this repository, the main programs are

1. city_list_from_county.py: Retrieve a list of cities within a US county, given a name for the county
2. county_pharmacy_list.py: Using the list of cities, get a list of pharmacies for each city and compile them into one list of the whole county.
3. get_top_pharmacies_for_county.ipynb: Given this list, determine how many of each type of pharmacy appears.  That is, what are the top five pharmacies in a region based on how many there are (as opposed to how popular one is).

All programs are written in Python.  Program 3. is a jupyter notebook and is the _main_ program for the script.
